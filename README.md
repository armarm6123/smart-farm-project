# Smart Farm Project

## โครงสร้างโปรเจกต์
- `1_mobile_app/`: ซอร์สโค้ดแอปพลิเคชันมือถือ (Flutter)
- `2_backend_api/`: ระบบ Backend API สำหรับเชื่อมต่อฐานข้อมูล
- `3_web_dashboard/`: เว็บไซต์สำหรับดูข้อมูลและจัดการฟาร์ม
- `4_database/`: สคริปต์โครงสร้างฐานข้อมูล (SQL)
- `5_hardware_esp32/`: โค้ดสำหรับบอร์ดควบคุม ESP32
- `6_python/`: ระบบ MQTT Bridge สำหรับส่งข้อมูลระหว่าง Hardware และ Server
- `6_user_test/`: ผู้ใช้งานสำหรับทดสอบระบบ
